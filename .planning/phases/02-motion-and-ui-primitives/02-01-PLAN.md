   - First line: `'use client';`

   - Import `useEffect, useState` from `'react'`
   - Export named function `LenisProvider` accepting `{ children }: { children: React.ReactNode }`

   ```typescript
     children: ReactNode;

     delay?: number;

   };
   ```

     - `whileInView="visible"`
     - `viewport={{ once: true, amount: 0.08 }}`
     - `transition={{ delay: shouldReduceMotion ? 0 : delay }}` — no delay when reduced motion active
