# bitremote-website

Marketing website for BitRemote (`beta.bitremote.app`).

## Development
- Install deps: `npm install`
- Run dev server: `npm run dev`
- Build static export: `npm run build` (outputs `out/`)

## Deploy
GitHub Actions deploys `out/` to GitHub Pages. The `public/CNAME` file is included for `beta.bitremote.app`.
